# NewsApp - MVVM

### JSON API from: https://newsapi.org/

## In the application: 
* Bulding complete programmatically (without storyboard),
* Using NSCache to cache images, 
* Using network API calls, 
* Using MVVM Pattern, 
* Creating custom ImageView with shadow.
